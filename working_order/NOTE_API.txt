




---------------Note API System
1) Api Create SO
- url api = (domain):(port)/web/api/create-so
- methods = post
- value = json
Example data ={	"reference": "your_customer_code_001",
				"journal_code": "your_short_code",
				"order_lines": [
					{
						"product_code": "your_product_code_001",
						"qty": 2,
						"price_unit": 1000
					},
					{
						"product_code": "your_product_code_002",
						"qty": 4,
						"price_unit": 2000
					},
					{
						"product_code": "your_product_code_003",
						"qty": 6,
						"price_unit": 3000
					}
				]
			}
Function note:
This API action will help you create the SO to your odoo system and run odoo
workflow function for help Your SO change state to paid. 
End then it will create the invoice and help you validate it.
